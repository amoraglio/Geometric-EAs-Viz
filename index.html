About

These visualizations demonstrate how distance metrics unify optimization algorithms across different representations. The geometric framework reveals that seemingly different algorithms share common underlying principles.

How to Use

  1. Click on any algorithm above to open its visualization
  2. Select a test function from the dropdown
  3. Adjust parameters using the control sliders
  4. Click "Start" to begin optimization
  5. Observe how the algorithm navigates while avoiding tabu regions
